# Hi, I'm Yusong

I study electrical engineering and information technology at TUM, and I build LLM-agent systems and the automation around them — multi-model orchestration, local fine-tuning, and the glue that makes an agent useful day to day.

A few things worth a look:

- **[Pantheon](https://github.com/yusongcao2004/pantheon-llm-roundtable)** — a Telegram bot where several LLMs discuss a question across rounds and then produce a neutral synthesis. Provider-adapter architecture, anti-sycophancy, per-discussion token accounting.
- **[Transformer Quest](https://github.com/yusongcao2004/transformer-quest)** — an interactive browser game that walks you through what happens inside a Transformer, one attention step at a time. [Play it in the browser.](https://yusongcao2004.github.io/transformer-quest/)
- **[TUM Moodle Archive](https://github.com/yusongcao2004/tum-moodle-archive)** — a privacy-first browser extension that archives your own course files locally, using your existing session and uploading nothing.
- **[claude-phone](https://github.com/yusongcao2004/claude-phone)** — a mobile web app plus a small FastAPI server that turns my phone into a remote control for Claude Code on my Mac: browse every past session, resume any of them, and watch results stream back live.

A recurring theme in what I build: the private data stays on my machine, and the model drafts while a person makes the call.
